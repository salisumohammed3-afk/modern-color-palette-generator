# Modern Color Palette Generator

A beautiful, modern web-based color palette generator with a dark theme.

## Features

- 🎨 Generate 5 random hex colors in a beautiful palette
- 📋 Click any color to copy its hex code to clipboard
- 🌙 Dark theme with gradient backgrounds
- 📱 Fully responsive design
- ⚡ Keyboard shortcut (spacebar) to generate new palettes
- 🎯 Smooth animations and hover effects

## Live Demo

Visit the live app: [https://salisumohammed3-afk.github.io/modern-color-palette-generator/](https://salisumohammed3-afk.github.io/modern-color-palette-generator/)

## How to Use

1. The app automatically generates 5 random colors on load
2. Click any color block to copy its hex code to your clipboard
3. Click "Generate New Palette" or press spacebar to create a new set of colors
4. Hover over colors to see their hex codes

## Technology

- Pure HTML, CSS, and JavaScript
- No external dependencies
- Modern CSS features (Grid, Flexbox, Gradients, Backdrop Filter)
- Responsive design with mobile-first approach

## Browser Support

Works in all modern browsers that support:
- CSS Grid
- Flexbox
- Clipboard API (with fallback for older browsers)
- CSS Gradients and Backdrop Filter
